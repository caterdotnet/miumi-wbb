>> What is Miumi? <<

Miumi is a virtual mascot that has an array of QoL, algorithms, and bots to use.
Miumi-wordbombbot (wbb) is the first bot added to its array.


>> What is Miumi-wbb? <<

Miumi-wbb (Miumi-wordbombbot) is a Word Bomb assistant bot. Give it a word slice (e.g. AW) and it instantly finds a valid word from a 286k+ word dictionary that
satisfies the slice. It can also auto-focus the game window and type the word for you — hands-free.

: Supports Roblox windows (Roblox Word Bomb) and Chrome windows (jklm.fun). More browsers will be supported soon.


>> How to use <<

1. Type your slice into the input box and press Enter or click Find Word.
2. The result appears in the Result box and in the Session History scrolling frame.


>> Settings <<

: Auto type — Miumi types the result into the focused window using driver-level keystrokes.
: Window focus — Miumi finds and activates Roblox or Chrome window before typing. If no matching window is found, it aborts and shows an error.
: Priority (poi) — Controls which word is picked from all valid matches:
  ; None (random): picks any valid word at random.
  ; Long words: always picks the longest matching word.
  ; Short words: always picks the shortest matching word.
  ; Hyphenated: prefers hyphenated words; falls back to random if none exist.
Only one poi mode can be active at a time."


>> Session history <<

Every word Miumi-wbb plays is logged in the Session History list.
Miumi will never repeat a word within the same session.
Click New Session to clear the list and start fresh.
Export List — saves the current session history to a .txt file. A save dialog lets you choose the filename and location.
